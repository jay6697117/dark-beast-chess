# 暗兽棋规则

## 🎲 基本设置
- **棋盘**：4×4方格（共16格）
- **棋子**：双方各8枚（共16枚），包含：
  - 动物：象、狮、虎、豹、狼、狗、猫、鼠
  - 颜色：红/蓝两色
- **初始状态**：所有棋子背面朝上随机摆放

## ⚔️ 吃子规则表
| 动物 | 可吃动物                         | 特殊规则               |
|------|----------------------------------|------------------------|
| 象   | 狮、虎、豹、狼、狗、猫           | 可被鼠吃               |
| 狮   | 虎、豹、狼、狗、猫、鼠           |                        |
| 虎   | 豹、狼、狗、猫、鼠               |                        |
| 豹   | 狼、狗、猫、鼠                   |                        |
| 狼   | 狗、猫、鼠                       |                        |
| 狗   | 猫、鼠                           |                        |
| 猫   | 鼠                               |                        |
| 鼠   | 鼠、象                           | 唯一可吃象的动物       |

## 🕹️ 游戏流程
1. **决定先后手**
2. **翻棋定色**：
   - 先手玩家必须随机翻开一枚棋子
   - 该棋子颜色即代表玩家阵营
3. **回合行动**（每回合二选一）：
   - ▶️ **翻棋**：翻开暗置棋子
     - 若为己方颜色则收入阵营
     - 若为敌方则归对方所有
   - ▶️ **走棋**：移动已翻开的己方棋子
     - 移动方向：上下左右相邻格
     - 遇敌时触发吃子判定

## ⚡ 特殊规则
- 🐭 **鼠克象**：鼠可吃象（"鼠钻象鼻"）
- ♟️ **同类互吃**：相同动物相遇时同归于尽
- 🔄 **防僵局**：重复走棋3次后，先手方需变招否则判负

## 🏆 胜利条件
- 消灭对方所有棋子
- 或迫使对方无法有效走棋

## 💡 策略提示
1. 优先记忆关键棋子位置（象/鼠）
2. 鼠是逆转战局的秘密武器
3. 中期开始需平衡翻棋与走棋
4. 保护大将避免被偷袭
